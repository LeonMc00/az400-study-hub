The % of deployments that cause a failure in production. Lower is better. One of the 4 DORA metrics.

Which branching strategy is best for continuous deployment teams?
Trunk-Based Development — all devs commit to main with short-lived feature branches (<2 days).
What is a monorepo?
A single repository that holds code for multiple projects or services. Easier to share code; harder to scale permissions.
What does a branch protection rule enforce?
Prevents direct pushes to main. Requires PRs, approvals, and passing checks before merging.
When should you use GitFlow?
When you have scheduled releases with multiple environments (dev → staging → prod) and need hotfix support.

What is the difference between CI and CD?
CI = auto build & test every commit. CD (Delivery) = always ready to deploy with human approval. CD (Deployment) = fully automatic to production.
What is a Blue/Green deployment?
Two identical environments. Blue = live. Green = new version. Flip traffic to Green. If it fails, flip back instantly. Zero-downtime rollback.
What does a pipeline agent do?
The machine that runs your pipeline jobs. Can be Microsoft-hosted (managed for you) or self-hosted (your own VM).
What is a Canary deployment?
Release to a small % of users first (e.g. 5%). Monitor for errors. Gradually increase traffic if healthy. Safe way to catch issues early.

How do you securely use a Key Vault secret in a pipeline?
Create a Variable Group in Azure DevOps linked to Key Vault. Reference the group in your YAML. The secret is never exposed in code or logs.
What is the difference between SAST and DAST?
SAST = Static — scans source code without running it. DAST = Dynamic — tests a running application by simulating real attacks.
What is least privilege in a pipeline context?
Service connections and agents should only have the permissions they actually need — nothing more. Reduces blast radius if compromised.
What does SCA scan for?
Software Composition Analysis — scans your open-source dependencies (npm packages, NuGet etc.) for known CVEs and licence issues.

What is the difference between SLI, SLO, and SLA?
SLI = the measurement (e.g. uptime %). SLO = your internal target (e.g. 99.9%). SLA = the contract with your customer.
What is an error budget?
The amount of downtime/errors you're allowed before breaching your SLO. If budget runs out, slow down releases until reliability improves.
What does Application Insights provide that Log Analytics doesn't?
Distributed tracing — it tracks a single request as it flows across multiple services, showing exactly where delays or errors occur.
What language do you use to query Log Analytics?
KQL — Kusto Query Language. Example: requests | where resultCode == 500 | summarize count() by bin(timestamp, 1h)

Secrets Management
  • Store secrets in Azure Key Vault — never hardcode in YAML
  • Reference via variable groups linked to Key Vault
  • Use managed identities instead of service principal secrets
  • Rotate secrets regularly — set expiry dates in Key Vault
  • Never log secrets — mask variables in pipeline output

Exam day strategy
🔥
Focus on Pipelines First
Over half the exam is Domain 3. If time is short, spend the majority of study hours here. Learn YAML pipeline syntax, agents, and deployment strategies cold.
🤚
Prerequisites Matter
You need AZ-104 (Admin) or AZ-204 (Developer) before earning the DevOps Expert cert. You can sit AZ-400 standalone, but the full cert requires a prerequisite.
🧪
Hands-On is Non-Negotiable
The exam has performance-based labs in a live Azure environment. You cannot pass on theory alone. Create a free Azure DevOps org and build real YAML pipelines.
💰
Exam Details
Cost: ~$165 USD. Passing score: 700/1000. Up to 5 attempts per year. 24hr wait after first fail, then 14 days between retakes. Renews annually via free online assessment.
🎯
Read the Whole Question
Questions have long case study scenarios. Key detail is often buried mid-paragraph. Watch for constraint words: "least privilege", "no downtime", "without modifying code".
🗝️
Secrets = Key Vault. Always.
If a question involves storing passwords, connection strings, or certificates — the answer is almost always Azure Key Vault. Know how to reference it from pipeline variable groups.
🕐
Suggested Study Timeline
Beginner: 12–16 weeks. Weeks 1–3: DevOps + Git fundamentals. Weeks 4–8: Pipelines deep-dive. Weeks 9–11: Security & IaC. Weeks 12–13: Monitoring. Weeks 14–16: Practice exams + labs.
